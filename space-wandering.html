Space Wandering

Overview

An interactive game built with Arduino and P5.JS

Duration – 2 weeks
Role – Designer, Programmer
Tools – Arduino, P5.JS
Team – Solo Everything!

Demo GitHub

Objective

Designing and creating compelling interactions that bridge the physical and digital using Arduino serial data and P5.JS as an integrated system.

Process

From Arduino serial data to P5.JS

I start by taking a simple circuit with two buttons and two LED bulbs; and connect it to P5.JS using serial data.

The process requires connection using three application platforms: Arduino IDE, P5 Serial Port, and Visual Studio Code; therefore settings across three applications need to be accurate and compatible.

Arduino IDE

P5.JS

From P5.JS to Arduino

In the next step, we are transmitting data from P5.js back to Arduino IDE and then triggering the physical LED light bulbs.

The function we use to communicate data is P5 write() which writes data to the PrinterWriter stream (serial port).


P5.JS

The Joystick

An Analog Joystick replaces two potentiometers and provides smooth axis-based controls for a better user experience in games.

Sending two integer numbers of x and y position, in P5.JS code, we split the data by a comma and map them with the P5 canvas.

Feedback

How to play the game?

The feedback from user testing session suggests that users are unclear on sensor controls.

The solution is to add some visual elements to P5.JS and a how-to-play instruction to get the user started for the game, making sure they know which hardware buttons/sensors to interact with.
